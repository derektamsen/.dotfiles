# dotfiles

All my Mac and Linux .&lt;files>

## Install
To install just run:
```sh
git clone https://github.com/derektamsen/.dotfiles.git ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```

## Configure System Settings and Packages
### Setup Mac OS X
```
script/setup_osx
```

### Setup Linux
```
script/setup_linux
```
